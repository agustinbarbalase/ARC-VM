# ARC Virtual Machine

![application-screenshot](./docs/)

The project is virtual machine for simulate ARC instruction set. It's based on *Principles of Computer Architecture - Miles J. Murdocca, Vincent P. Heuring - Prentice Hall (1999)*, and the project is inspired by [Virtual Machine in C](https://web.archive.org/web/20200121100942/https://blog.felixangell.com/virtual-machine-in-c).
